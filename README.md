 📊 SuperStore Sales Analysis Dashboard (Power BI)

 📌 Overview
This project presents an advanced Business Performance Dashboard built using Power BI to analyze sales, profit, and operational efficiency using the SuperStore dataset.



 🚀 Key Features
- 📈 Sales, Profit & Profit Margin Analysis  
- 🔄 Year-over-Year (YoY) & Month-over-Month (MoM) Growth  
- 🏆 Top & Bottom Product Analysis  
- 🌍 Region-wise Performance Insights  
- 👤 Manager-wise Performance & Efficiency Analysis  
- 🔁 Return Analysis to identify loss areas  
- 🎯 KPI Indicators with dynamic color formatting  
- 🧭 Interactive dashboard with slicers and drill-through  


 📷 Dashboard Preview
[Dashboard Screenshots](SuperStore-Sales%20Dashboards.zip)


 🔧 Tools & Technologies
- Power BI  
- DAX (CALCULATE, FILTER, SAMEPERIODLASTYEAR, PREVIOUSMONTH)  
- Data Modeling (Star Schema)  
- Data Visualization  



 📊 Data Model
 **Orders Table** → Main fact table containing sales and profit data  
 **Returns Table** → Used to analyze returned orders and loss impact  
 **People Table** → Used for manager-level performance analysis  



 📈 Key Insights
- High sales does not always result in high profitability  
- Certain products and regions contribute to higher return rates  
- Manager-level performance varies significantly based on efficiency  
- Return analysis highlights hidden loss areas in the business

 🎯 Business Impact
This dashboard helps stakeholders:
- Identify top and bottom performing products  
- Analyze growth trends using YoY and MoM metrics  
- Detect inefficiencies caused by returns  
- Evaluate manager performance for better decision-making  



 🧠 Conclusion
This project demonstrates how data visualization and DAX can be used to transform raw data into meaningful business insights and support data-driven decisions.
